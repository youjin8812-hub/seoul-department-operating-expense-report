# 분석 기준

## 집계 단위

```
해당년도 → 해당월 → 전체부서명 → 비목 → 집행금액 합계
```

분석 단위는 라인아이템(집행 건) 기준입니다. 문서 1건에 여러 라인아이템이 딸린 구조(문서 단위와 별개)이므로, "건수"를 인용할 때는 항상 라인아이템 기준임을 명시합니다.

## 부서 비교 조건

- 부서 간 순위·구조 비교는 **동일 연도·동일 월(범위)·동일 비목 기준**을 충족하는 부서끼리만 수행합니다.
- 이 프로젝트의 데이터는 2026년 1~6월 단일 구간이므로, "동일기간"은 곧 "6개월 전 기간 데이터를 보유했는가"로 판정합니다.
- 6개월 전 기간을 보유하지 못한 부서는 순위·구조 비교 대상에서 제외하고 **"비교 불가"**로 표시합니다. 임의로 다른 기간과 비교하지 않습니다.

## 집중도 지표와 부서 비교의 구분

- 상위N 부서 점유율, HHI(허핀달-허쉬만 지수) 등 **집중도 지표는 전체 부서를 대상으로 한 지출 분산도** 지표이며, 위의 "동일기간 부서 비교"와는 목적과 대상 범위가 다릅니다.
- 두 지표를 같은 표·차트에 섞어 쓰지 않고, 항상 대상 범위(14개 동일기간 부서 vs 136개 전체 부서)를 명시합니다.

## 고액 집행 표현 원칙

- 통계적으로 상위권에 있는 집행 건은 **"고액 집행 후보"**로만 표현합니다.
- IQR(사분위범위) 기준과 상위 5% 기준을 구분해 제시하며, 두 기준의 건수가 다를 수 있음을 명시합니다.
- 고액이라는 사실 자체를 오류·부적정 집행으로 단정하지 않습니다. 원인·의도·적정성은 데이터로 확인되지 않는 한 서술하지 않습니다.

## 반복 패턴 표현 원칙

- 동일 부서·비목·금액 조합이 반복되는 경우 반복 횟수와 반복이 발생한 월수를 사실로만 제시합니다.
- "정례적", "관행적" 등 반복의 사유나 성격을 규정하는 표현은 사용하지 않습니다.
- "동일 금액 반복"(부서·비목·금액 조합 일치)과 "동일 집행 건 반복"(모든 필드가 완전히 일치하는 중복행)은 서로 다른 개념이며, 후자가 0건으로 확인된 경우 그 사실을 명시합니다.

## 데이터 부족·품질 이슈 표현

- 결측·형식 오류·비교 불가 상황은 임의로 보완하지 않고 다음 셋 중 하나로만 표시합니다.
  - **데이터 확인 필요**: 특정 필드가 결측이거나 형식이 불명확한 경우
  - **비교 불가**: 비교 대상 간 조건(기간 등)이 다른 경우
  - **집계 대상 없음**: 정의상 해당하는 데이터가 존재하지 않는 경우

## 개인정보 처리

- `전화번호`, `작성자` 컬럼은 모든 분석·집계·시각화·보고서에서 전량 제외합니다.
- `집행대상` 컬럼은 원문을 어떤 산출물에도 노출하지 않으며, 반복 횟수 등 집계값으로만 내부 분석에 사용합니다.
- 인터랙티브 대시보드는 위 3개 컬럼을 애초에 임베드 데이터에 포함하지 않아, 브라우저에서 소스를 열어봐도 노출되지 않도록 설계했습니다.
- 모든 산출물은 생성 시 원본 데이터와 대조해 개인정보 노출 여부를 스크립트로 자동 검증합니다(`scripts/validate_data_and_privacy.py`).
