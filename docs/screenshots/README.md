## 추가로 캡처하면 좋은 파일 (선택)

| 파일명 | 내용 | 캡처 방법 |
|---|---|---|
| `dashboard-filter-example.png` | 대시보드에서 특정 월·부서를 필터링해 KPI와 차트가 갱신된 상태 | 필터 조작 후 캡처 |
| `hwpx-preview.png` | `output/*.hwpx`를 한컴오피스 한글에서 열어 1페이지 화면 | 한글에서 열고 캡처(한컴오피스 필요) |

## 캡처 시 주의사항

- 개인정보(전화번호·작성자·집행대상 원문)나 로컬 절대경로가 화면에 노출되지 않았는지 캡처 후 반드시 육안 확인합니다. 이 프로젝트의 산출물 자체에는 해당 정보가 없지만, 프로그램 창의 파일 경로 표시줄 등에 로컬 사용자명이 노출될 수 있으므로 창 제목표시줄·경로 표시 영역은 잘라내는 것을 권장합니다.
- 새 스크린샷을 추가하면 `README.md`의 "실행 화면" 섹션에 실제 파일명으로 링크를 추가하세요. 존재하지 않는 파일은 링크하지 않습니다.
